Statlet: Exploring Probabilities—Build a Slot Machine

Note:

This statlet covers probability. The goal is to gain intuition about probabilities, probability distributions, and how probabilities combine into joint probabilities. The dataset is an excerpt from the 2013 causes-of-death counts across all US States (Center for Disease Control).

Toward the end of this statlet you will build a slot machine that allows you to bet on one or more causes of death.
